#  Torchify

> *"Because darkness shouldn't be free."*

Torchify is a satirical startup website that pokes fun at today's subscription economy. Turning your flashlight **ON** is completely free—but turning it **OFF**? That's a premium feature.

Built as a static web project using **HTML**, **CSS**, and **JavaScript**, Torchify combines clean UI design with playful interactions and a professional GitHub workflow.

---

##  Features

- Interactive flashlight built with HTML, CSS, and JavaScript
- Premium-only "Turn OFF" experience
- Dedicated pricing section with satirical subscription plans
- Elegant, handcrafted user interface
- Responsive layout for desktop, tablet, and mobile
- Smooth animations and hover effects
- Well-documented repository with contribution guidelines and templates

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
