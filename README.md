# Torchify

> The world's first subscription-based flashlight.

Turn your flashlight on for free.

Turning it off? That's a premium feature.

---

## About

Torchify is a satirical startup landing page built using HTML, CSS, and JavaScript. It parodies the modern subscription economy by allowing users to switch a flashlight on for free while requiring a premium subscription to switch it off.

## Features

- Responsive landing page
- Interactive flashlight
- Premium subscription modal
- Startup-inspired UI
- Smooth animations

## Tech Stack

- HTML5
- CSS3
- JavaScript
